# cljdropbox

cljdropbox implemented Clojure

## Usage

1. Create info.env file to project root

2. input access-token in info.env

   {:access_token "1234567889"}

3. lein run

4. use function

<code>
   (search "" "*.txt")
   (download "/test.txt" "test.txt")
   (display-usage)
</code>
